# Image Encryption With Chaotic Maps

## Overview
This project focuses on **evaluating the efficiency of different chaotic maps** in the context of image encryption.

Instead of implementing just one encryption method, the system compares multiple chaotic map algorithms to analyze their effectiveness in terms of randomness, security, and performance when applied to digital images.

The goal is to study how well chaotic systems can be used for **secure image processing** and which maps provide better encryption characteristics.

---

## Features

- **Chaotic Map Evaluation**
  - Compare multiple chaotic maps
  - Analyze their behavior in encryption scenarios

- **Image Encryption Testing**
  - Apply chaotic maps to encrypt images
  - Evaluate how well image data is obscured

- **Performance Analysis**
  - Measure efficiency and execution performance

- **Statistical Analysis**
  - Examine randomness and distribution of pixel values
  - Evaluate encryption strength through metrics

- **Image Processing**
  - Handle and manipulate image data for testing
  - Support for grayscale images

---

## Technologies Used

### Programming Language
- **Python**

### Libraries
- **NumPy** – numerical computations and matrix operations  
- **OpenCV (cv2)** – image processing  
- **Matplotlib** – visualization and analysis  

---

## How It Works

1. Load an input image  
2. Apply different chaotic map algorithms  
3. Encrypt the image using each method  
4. Evaluate results based on: 
   - Randomness  
   - Visual distortion
   - Safety
